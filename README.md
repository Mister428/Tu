NOIR ACHATS

Bienvenue sur NOIR ACHATS, une boutique digitale futuriste développée par CHAPEAU NOIR ✴💥.


---

🌐 Description

NOIR ACHATS est un site de vente de produits digitaux avec :

Produits horizontaux et grands

Panier complet avec calcul du total

Commande directe sur WhatsApp

Paiement via Orange Money

IA photo pour générer des images à partir d'un texte

Galerie futuriste avec rendu premium

Animations et effets cyberpunk (Matrix, néon, glow)

Date et heure en temps réel


Le site est entièrement responsive et pensé pour un déploiement rapide.


---

1. Cloner le dépôt GitHub

git clone https://github.com/<ton-utilisateur>/noir-achats.git
cd noir-achats
